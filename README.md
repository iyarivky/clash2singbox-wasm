# clash2singbox-wasm
Stupid attempt to convert clash2singbox go binary to WebAssembly binary

## Usage
- Go to https://c2s.deno.dev
- Input your Clash YAML config
- Press "Convert" button
- Done

## Support
- shadowsocks （仅包含 v2ray-plugin, obfs 和 shadow-tls 插件）
- shadowsocksR
- vmess
- vless (含 reality)
- trojan
- socks5
- http
- hysteria
